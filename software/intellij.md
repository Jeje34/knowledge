# Intellij

## Shortcuts

**CTRL + SHIFT + F** : find in path

**CTRL + ALT + B** : go to implementation

**SHIFT x2** : search everywhere

**CTRL + /** : comment a line

**CTRL + SHIFT + /** : comment a block

**CTRL + T** : update

**CTRL + K** : commit
 
**CTRL + C** : select current line (if nothing already selected)

**ALT + F7** : find usages
 

### Debug

**F8** : go to next line
