# jQuery

## Show / hide an element

    $( ".target" ).hide();
    $( ".target" ).show();
    $( ".target" ).toggle(); // cache si affiché, affiche si caché
    
## Add / remove a class

    $( "p" ).removeClass( "myClass noClass" );
    $( "p" ).addClass( "yourClass" );
    
## Check if an element has a given class
    
    $( "#mydiv" ).hasClass( "foo" )
